# Решение задачи о рюкзаке

Данная программа занимается решением "задачи о рюкзаке". Для этого реализовано два алгоритма: жадный и метод ветвей и границ.

## Скриншот

![Вывод программы](https://imgur.com/a/2NpfZNl)

## Стек технологий:

C++17
CMake 3.15

## Компиляция и запуск:

Для CMake:
Распаковать архив с исходным кодом. Создать папку build. Открыть терминал в папке build. Написать следующие команды:
`cmake ..`
`cmake --build .`
Перейти в bin. Запустить файл backpack.exe.

Для gcc:
Открыть терминал в папке src и прописать следующую команду:
`g++ -std=c++17 main.cpp items.cpp node.cpp -o backpack`

## Функционал:

- Нормальное или равномерное (на выбор) распределение значений для весов.
- Равномерное распределение ценностей в диапазоне от 0 до X.
- Расчёт веса и ценности рюкзака собранного жадным алгоритмом
- Расчёт ценности рюкзака собранного методом ветвей и границ
- Относительная погрешность для жадного алгоритма
